# Blogpost2Audio

## 📦 Initial Setup
1. Copy .env file (`cp .env.example .env`) and add the necessary info
2. Run `npm install`
3. Build the frontent (`cd frontend && npm i && npm run build`)
  
  
## 🚀 Start App
`npm run start` 
  
  
  
## ⚙️ Deployment
This App can be deployed to IBM Cloud using cloud foundry.
